# GP Articles — Training Corpus

A curated collection of 30 opinion and analysis articles for A-Level General Paper (Syllabus 8881) training.

## Structure

- **30 articles** from CNA, The Guardian, and BBC Future
- **11 themes** covering the full GP syllabus
- **8 regions** including Singapore, Asia, Europe, Africa, Middle East, Latin America
- **~98,000 words** of training material

## Usage

Open `index.html` in a browser, or deploy as GitHub Pages:

1. Push to GitHub
2. Enable GitHub Pages in repo Settings → Pages
3. Set source to `main` branch, root directory
